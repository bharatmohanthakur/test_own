Stop asking "which option?" — just pick the best one and DO IT.
**Why:** User wants autonomous execution, not a menu of choices. Every question wastes time.
**How to apply:** When multiple approaches exist, evaluate them, pick the best, and execute. Only stop for: critical errors, need credentials, or budget approval for expensive operations (>$20). Report results, not options.
